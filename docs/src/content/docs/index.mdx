## Quick Install

```bash
bun add binja
```

## Quick Example

```typescript
import { render, compile } from 'binja'

// Runtime mode (development)
const html = await render('Hello, {{ name|upper }}!', { name: 'world' })
// Output: Hello, WORLD!

// AOT mode (production) - 160x faster
const template = compile('<h1>{{ title }}</h1>')
const html = template({ title: 'Welcome' })
// Output: <h1>Welcome</h1>
```
