## Why binja?

<CardGrid>
  <Card title="2-4x Faster Runtime" icon="rocket">
    Pure TypeScript implementation optimized for Bun. Consistently outperforms Nunjucks in every benchmark.
  </Card>
  <Card title="160x Faster AOT" icon="lightning">
    Pre-compile templates to JavaScript functions for maximum production performance.
  </Card>
  <Card title="Multi-Engine Support" icon="puzzle">
    Jinja2, Handlebars, Liquid, and Twig - all through a unified API with shared filters.
  </Card>
  <Card title="84 Built-in Filters" icon="setting">
    Complete Jinja2 + Django filter set. String, number, date, list, URL, and more.
  </Card>
</CardGrid>

## Quick Install

```bash
bun add binja
```

## Quick Example

<Tabs>
  <TabItem label="Runtime Mode">
    ```typescript
    import { render } from 'binja'

    // Simple and fast for development
    const html = await render('Hello, {{ name|upper }}!', { name: 'world' })
    // Output: Hello, WORLD!
    ```
  </TabItem>
  <TabItem label="AOT Mode">
    ```typescript
    import { compile } from 'binja'

    // Compile once at startup
    const template = compile('<h1>{{ title }}</h1>')

    // Use many times - sync and extremely fast
    const html = template({ title: 'Welcome' })
    // Output: <h1>Welcome</h1>
    ```
  </TabItem>
  <TabItem label="Environment">
    ```typescript
    import { Environment } from 'binja'

    const env = new Environment({
      templates: './views',
      cache: true,
    })

    // Render template files with inheritance
    const html = await env.render('pages/home.html', {
      title: 'Home',
      user: { name: 'John' }
    })
    ```
  </TabItem>
</Tabs>

## Performance

|-----------|-------|----------|---------|
| Simple Template | 371K ops/s | 96K ops/s | **3.9x** |
| Complex Template | 44K ops/s | 23K ops/s | **2.0x** |
| HTML Escaping | 985K ops/s | 242K ops/s | **4.1x** |

## Framework Adapters

<Tabs>
  <TabItem label="Hono">
    ```typescript
    import { Hono } from 'hono'
    import { binja } from 'binja/hono'

    const app = new Hono()
    app.use(binja({ root: './views' }))
    app.get('/', (c) => c.render('index', { title: 'Home' }))

    export default app
    ```
  </TabItem>
  <TabItem label="Elysia">
    ```typescript
    import { Elysia } from 'elysia'
    import { binja } from 'binja/elysia'

    const app = new Elysia()
      .use(binja({ root: './views' }))
      .get('/', ({ render }) => render('index', { title: 'Home' }))
      .listen(3000)
    ```
  </TabItem>
</Tabs>
